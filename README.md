# Smatrix::Io

See [My blog post about sparse matrices][http://educated-guess.com/2011/02/22/sparse-matrices/]

## Installation

Add this line to your application's Gemfile:

    gem 'smatrix-io'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install smatrix-io
